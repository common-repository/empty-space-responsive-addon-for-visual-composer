Visual Composer Extension that will allow user to give responsive space across all devices
== Description ==
<h3>Features</h3>

With this addon, user can customize the vertical space height, respectively to different devices. This plugin is an advanced version of default empty space addon in visual composer plugin.

== Installation ==
<b>Note:</b> <b><i>You require <a href="http://codecanyon.net/item/visual-composer-page-builder-for-wordpress/242431?ref=thyash" target="_blank">Visual Composer</a> Installed to use this add-on.</i></b>

1. Go to plugins in your dashboard and select 'add new'
2. Search for 'Empty Responsive Space Addon for Visual Composer' and install it
3. Go to Pages or Posts and add "Empty Responsive Space" in content section in your pages.
